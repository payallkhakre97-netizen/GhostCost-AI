# 🛡️ GhostCost AI: Global Ethical Auditor
**Revealing the invisible price of consumption.**

### 🚀 Features
- **Multimodal Audit:** Image & Text based product scanning.
- **Agentic Reasoning:** Analyzing Human, Animal, and Environmental costs.
- **Sustainability Switch:** Suggesting Bio-enzymes and eco-alternatives.

### 🛠️ Tech Stack
